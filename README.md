# Calculation-of-Section-Properties-for-Arbitrarily-Shaped-Cross-Sections
Python program to calculate section properties of a Arbitrary shaped cross section
This Python program utilises greenns theorem to calculate setion properties of any type of cross-section. It can be I section T section, circular section, angle section, builtup sections, etc.
Reference
K.F. Chung, (1993), "The state-of-the-art of section property calculation of structural members with arbitray shape", The Steel Construction Institute, Silwood Park, Ascot, Berkshire SL5 7QN, UK
"https://doi.org/10.1016/0143-974X(95)93169-5"
